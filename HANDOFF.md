### 5. 审核中心

- 新增独立页面：`#/review-center`
- tab 列表当前只有 `视频`，后续可扩展。
- 展示所有账户上传的视频，包括：
  - 待审核
  - 审核不通过
  - 已发布
  - 已下架
- 视频列表字段：
  - 视频名称
  - 视频封面
  - 点赞数量
  - 发布时间
  - 状态
  - 操作
- 点击视频封面可预览视频。
- 发布时间以审核通过时间为准；没有发布时间时展示 `--`。
- 状态筛选是右侧下拉框：
  - `全部`
  - `待审核`
  - `审核不通过`
  - `已发布`
  - `已下架`
- 默认筛选为 `全部`。

审核中心操作列固定展示两个按钮：

- `下架 / 上架`
- `审核`

不可用的操作会置灰并禁用。

状态规则：

- `待审核`：可执行 `审核`。
- `审核不通过`：两个操作都置灰。
- `已发布`：可执行 `下架`。
- `已下架`：可执行 `上架`。

审核操作：

- 点击 `审核` 打开审核弹窗。
- 可选择：
  - `审核通过`
  - `审核不通过`
- 审核通过时写入当前时间作为发布时间。

上下架操作：

- 点击 `下架` 有二次确认。
- 下架后状态展示为 `已下架`，操作变为 `上架`。
- 点击 `上架` 有二次确认。
- 上架后状态恢复为 `已发布`。

## 当前数据说明

视频数据仍是纯前端本地模拟数据，保存在 `localStorage`：

- key：`phanty-movie-videos-state`

默认数据中包含多个状态：

- 已发布
- 已下架
- 待审核
- 审核不通过

为了兼容旧浏览器缓存，`script.js` 中会自动补齐新增示例数据。

## 部署方法

本项目线上地址：

- `https://phantymovie-canvas-demo.vercel.app`

部署方式：

- Vercel 已连接 GitHub 仓库。
- 推送到 `origin/main` 后，Vercel 会自动从 GitHub `main` 分支部署。
- 通常等待 1-2 分钟后刷新线上地址即可看到更新。

本项目当前没有 `.openai/hosting.json`，也没有本地构建步骤；页面是静态文件，部署依赖 GitHub push 触发 Vercel 自动部署。

## 注意事项

- 不要提交：
  - `.DS_Store`
  - `assets/.DS_Store`
- 项目是纯静态原型，没有后端接口。
- 视频上传使用浏览器本地 `FileReader` / `URL.createObjectURL` 模拟。
- 审核、上下架、发布状态都保存在浏览器 `localStorage` 中。
- 如果需要重置演示数据，可在浏览器中清理站点 localStorage。
